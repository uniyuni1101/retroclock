# retroclock
味気ないターミナルを時計へと変化させるコマンド

# 特徴
複数の時計のテーマを表示可能。


# 使い方
```
$ retroclock -h
Usage retroclock [option]
    -d --delay int
        Set the delay between updates (msec)
    -t --theme string
        Set the display clock theme
    -h --help
        Show retroclock help
    -v --version
        Show retroclock info
```
